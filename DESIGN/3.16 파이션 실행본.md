# 🔥 핵심 실행 흐름 (절대 변경 없음)

```text
run.py (while True)

6. 중복 체크 (이미 포지션 있으면 금지)
7. 리스크 검사 (risk_gate)
8. 안전 주문 (order_manager)
9. 체결 확인 (fill_tracker)
10. 상태 업데이트
11. 청산 실행 (exit_engine)
12. 상태 초기화
```
